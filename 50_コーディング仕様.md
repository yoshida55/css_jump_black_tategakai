# ストアサイト（和菓子）コーディング仕様

## 基本設定

| 項目 | 値 |
|------|-----|
| コンテンツ幅 | 1024px（header・footer・Google Mapは全幅） |
| ブレイクポイント | 768px |
| フォント | Google Fonts「Noto Serif JP」 |

---

## 各セクション仕様

### Header
- 背景画像を設定し、その上にロゴ・ナビ・ボタンを配置
- 背景画像はPCとSPで切り替え（`<picture>`タグ or メディアクエリ）
- **縦書きレイアウト**：`writing-mode` を使用
- ロゴは `h1` タグ
- グローバルナビは `nav > ul > li` 構成
- 「オンラインストアを見る」ボタン
  - `position` で左下に配置
  - `background` で透過
  - 内側に白枠の線を引くため、外側を `div` で囲む

### News（お知らせ）
- `dl` `dt` `dd` タグでマークアップ
- PC表示：5件表示
- SP表示：3件表示（4件目以降は非表示）
  - `nth-of-type(n + 4)` で4件目以降を `display: none`
- タイトルは `h2` タグ
  - 英語・日本語をそれぞれ `span` で囲み、ブロック化

### Products1（商品のご紹介1）
- 画像とテキストエリアを `position` で重ね合わせ
- 幅は `%` 指定（画面幅に応じて伸縮）
- テキストエリアは `background` で透過
- タイトルは `h2` タグ（英語・日本語を `span` で分離）
- **SP表示時**：タイトルの英語部分を非表示

### Products2（商品のご紹介2）
- 基本はProducts1と同じ構成
- コンテンツの並びが逆 → `writing-mode` を変更して並び順を切り替え

### Shop（店舗のご案内）
- Google Map埋め込み（`iframe`）
- 幅：全幅（100%）
- Mapの下にできる隙間は `vertical-align: bottom` で解消

### Footer
- 背景色：黒
- 左右のコンテンツを Flexbox で両端揃え
- 幅：全幅

---

## レスポンシブ対応

```css
/* PC */
@media (min-width: 769px) {
  /* PC用スタイル */
}

/* SP */
@media (max-width: 768px) {
  /* SP用スタイル */
}
```

---

## 縦書きレイアウトのポイント

```css
/* 縦書き設定 */
writing-mode: vertical-rl; /* 右から左へ縦書き */
```

- ブラウザごとに挙動が異なるため、Chrome・Safari・Firefox で表示確認必須
- 縦書き時は `margin` `padding` の方向に注意（上下左右が入れ替わる）
